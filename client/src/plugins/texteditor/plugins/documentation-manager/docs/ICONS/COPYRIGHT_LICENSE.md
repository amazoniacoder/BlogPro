## Technical Specifications

### BlogPro Icon Standards
All BlogPro original icons follow these specifications:

```svg
<!-- BlogPro Icon Template -->
<svg xmlns="http://www.w3.org/2000/svg" 
     width="20" height="20" 
     viewBox="0 0 24 24" 
     fill="none" 
     stroke="currentColor" 
     stroke-width="2" 
     stroke-linecap="round" 
     stroke-linejoin="round">
  <!-- Icon content -->
</svg>
```

**Standards:**
- **Grid System:** 24x24px artboard
- **Stroke Width:** 2px consistent
- **Style:** Outline/stroke (not filled)
- **Corners:** Round caps and joins
- **Theming:** Uses `currentColor` for CSS control
