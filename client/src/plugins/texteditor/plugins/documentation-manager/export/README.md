# documentation-manager

BlogPro plugin with integrated UI system

## Installation

```bash
npm install @blogpro/documentation-manager
```

## Usage

```javascript
import { loadUISystem } from '@blogpro/documentation-manager/ui-system/ui-loader.js';

// Load UI system
const uiSystem = loadUISystem();
await uiSystem.styles();
```

## Features

- ✅ Integrated BlogPro UI System
- ✅ Minimalist design components
- ✅ Dark/light theme support
- ✅ Mobile responsive
- ✅ TypeScript support
